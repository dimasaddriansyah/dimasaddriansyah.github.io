Project Overview

A project that aims to create a landing page for a bag store application website. This start page will be used to provide a brief and interesting description of the bag store application to website visitors. The purpose of making this landing page is to attract the attention of visitors and turn them into potential customers for the bag store application. Therefore, this landing page must be designed to be visually attractive, easy to understand, and have sufficient information to motivate visitors to learn more about the bag store application. Some of the features that are expected to be found on this landing page are a brief description of the bag store, advantages, the latest products, and customer testimonials.

LoFi Design

In making a landing page, need an initial structure or description for positioning and elements that will be used later. These elements include the header, footer, hero section, CTA button, why choose us, our features, new product, our story, and also subscriptions.

HiFi Design

After creating a wireframe, then create a visual appearance so that it can attract users to be able to see it, this visual appearance is very important and detailed, in choosing colors, typography, photos and others. It is hoped that this creation can communicate the form of visual design to users who see it

Design System

To make it easier to work on this landing page and save time, a design system is needed which includes color, typhography, and also several component cards.
